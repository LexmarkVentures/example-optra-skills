# Control Panel

In order to configure network settings on the device, there is an embedded webserver hosted on the device called the control panel. The control panel can be accessed in two ways: monitor and keyboard or remotely through the portal.

If a user plugs in a monitor without a skill running on device, the control panel login page will show on the screen. If the device is already on the network and enrolled, a user can click on the device in the portal and in the overview tab there should be a "Control Panel" button in the bottom right. Once this is pressed a tunnel will open to the control panel remotely.

Both of these methods will bring you to a page that asks for a password. By default, this password is the serial number of the device. Once logged in, the user will have access to configure the wifi, network proxies, and dns addresses.
